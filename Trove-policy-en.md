## 1. Core Architecture and Server Declaration
- This application is a purely offline, server-free, zero internet-dependent utility application. The core architectural rules are solemnly declared as follows:
- This application does not build, operate, rent, or use any self-owned or third-party application servers. It has no background network request modules, no active network communication capability, and all functions can be used in full in a completely offline state.
- This application has no account registration or login system. You can use all core functions without providing any identity information, and there is no user data collection behavior based on accounts.
The only cloud interaction involved in this application is the data synchronization operation completed by the native iCloud framework of Apple's iOS system, only after you actively and manually enable the Apple iCloud synchronization function. This process is independently controlled and executed entirely by Apple Inc. The application itself cannot intercept, read, upload, or tamper with any data in this process, nor does it participate in any link of data transmission.
## 2. Absolute Zero Data Collection Declaration for Data Collection and Use
We hereby make an irrevocable solemn commitment: this application will not actively collect, record, cache, upload, share, or sell any of your personal information, device information, collection content data, and usage behavior data in any scenario or any functional link. The specific rules are as follows:
- Collection Content Data: All collection-related data you record, edit, and store in this application (including but not limited to collection name, category, specification, serial number, purchase information, valuation, notes, images, tags, classification, and all other content) is only encrypted and stored in the local storage of your device. We cannot view, access, obtain, use, or disclose such data in any form, nor will we conduct any algorithmic analysis or commercial utilization of such data.
- Device and Personal Identity Information: This application will not collect any device information and usage behavior data, including your device identifier, IP address, operating system version, hardware information, usage duration, operation behavior, click path, etc. It will not require or induce you to provide any personal identity information such as name, email address, phone number, and mailing address.
- Sole Exception Statement: The only possible information interaction involved in this application is the system-level data processing generated by the native mechanism of Apple's iOS system for iCloud synchronization. This process is independently operated entirely by Apple Inc. For the relevant data processing rules, please refer to Apple's official privacy policy. We do not participate in any link of this process, nor can we obtain or retain any information therein.
## 3. Data Storage and Synchronization Rules
All your application data is only stored in the following two locations, 100% independently controlled by you, and we have no access or management rights:
### 3.1 Local Device Storage
- All your data is only encrypted and stored in the local storage space of your iOS device by default. All data reading and writing, editing, querying, and processing operations are completed offline locally on the device, and never leave your device throughout the process.
- When you uninstall this application, all application data stored locally on the device will be automatically and permanently deleted, and we cannot perform any form of data recovery operation.
### 3.2 iCloud Cloud Synchronization (Only Effective When Actively Enabled by the User)
- Only when you manually enable iCloud synchronization permission for this application in the iOS system settings, the iOS system will synchronize your local data to your personal exclusive Apple iCloud account storage space.
- This synchronization process is executed entirely by the native iCloud framework of Apple's iOS system. We cannot access, view, download, modify, or disclose any data you store in iCloud, nor shall we bear the operation and security responsibilities of the iCloud service itself. The privacy and security rules of the iCloud service are subject to the official statements and privacy policy issued by Apple Inc.
- You can turn off the iCloud synchronization permission in the iOS system settings at any time. After turning it off, the system will immediately stop the cloud synchronization operation, and it will not affect the normal use of your local data. You can manage and delete all data synchronized by this application to iCloud through the iOS system iCloud settings at any time.
## 4. System Permission Application and Usage Instructions
This application strictly follows the "minimum necessary" principle to apply for system permissions, and will only apply for the necessary permissions from the iOS system when you actively use the corresponding functions. All permissions are optional authorizations, and you can turn off the corresponding permissions in the system settings at any time, which will not affect the normal use of the basic core functions of the application after closing. All data related to permissions is only processed locally on the device, and will never be uploaded, cached, or shared with any third party. The specific instructions are as follows:
- Photo Library Permission: Only used for you to select collection-related images from the system photo library and store them in the local database of this application. This application only reads the image files you actively select, and will not read or upload any other content in your photo library.
- Camera Permission: Only used for you to take collection-related images and store them in the local database of this application. The captured images are only stored locally on the device, will not be uploaded to any cloud, and will not be saved in the system photo library (unless you actively save them).
- This application will not apply for or obtain any system permissions unrelated to the core function of collection recording, including your location, contacts, microphone, Face ID/Touch ID, motion and health, etc. If the subsequent version adds new functions and needs to apply for additional permissions, this privacy policy will be revised simultaneously, and your active consent will be obtained in advance when you use the corresponding functions.
